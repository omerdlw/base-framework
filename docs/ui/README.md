# UI Elements

Bu klasor UI elementlerinin dokumantasyonunu icerir.

## Standart Stil Yaklasimi

Tum elementlerde stil girisi icin tek standart prop kullanilir: `className`.

- Basit elementler (`Button`, `Input`, `Textarea`) icin `className` string verilebilir.
- Karmasik elementler (`Select`, `Popover`, `Tooltip`) icin `className` slot bazli object olarak verilebilir.
- Geriye donuk uyumluluk icin eski `classNames`/`externalClasses` kullanimlari halen calisir.
